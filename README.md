# DoseCerto - Aplicativo de Dosagem Médica
# Unipar ADS Casacavel - Alunos: Rodrigo, Amilton, Kevyn e Fabricio
# Trab de Laboratório de Software II - Prof: Luiz Fernando Rodrigues da Silva

O DoseCerto é um web app utilitário mobile-first voltado para a área da saúde. Seu objetivo é realizar a conversão matemática exata de dosagens de medicamentos injetáveis, da dose prescrita para o volume a ser extraído. E marcações de medicamento.
